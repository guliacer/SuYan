## 📸 界面与用法演示

> 下面动图都来自 `photo/`，按日常使用顺序排好。每张图下面有简短说明，方便对照着看。

### 1. 本地素材导入

从电脑里批量选图，很快就能建好本地素材库。

<img src="./photo/本地素材导入.gif" alt="本地素材导入" width="860" />

**动图里在做什么：**

1. 左侧点「导入素材」→「导入图片」
2. 在文件对话框里多选本机图片
3. 确认后自动入库，瀑布流里会出现缩略图
4. 也可以直接粘贴剪切板图片，或导入 Word 里的图文提示词

刚装好、手头已有一批效果图时，从这里开始最合适。

### 2. 网络素材导入

粘贴分享链接，把远程素材下载到本地库。

<img src="./photo/网络素材导入.gif" alt="网络素材导入" width="860" />

**动图里在做什么：**

1. 在「导入素材」里选网络 / 分享链接导入
2. 粘贴别人或平台发来的链接
3. 软件解析后下载图片和关联提示词，写入本机
4. 若站点要代理，可先到「网络代理」里配好再试

适合从社区、同事或分享平台搬素材，不用一张张另存为。

### 3. 分享与导入

把自己的素材打成包发给别人，也能导入对方发来的 ZIP。

<img src="./photo/分享与导入.gif" alt="分享与导入" width="860" />

**动图里在做什么：**

1. **导出：** 详情页点「分享」，或在「批量管理」里勾选多组后导出 ZIP
2. 包里带效果图和对应提示词，换电脑、换人都能传
3. **导入：** 收到 ZIP 后，用「导入分享」一键入库
4. 导入后还能继续改、打标签、再分享

不靠账号，用文件就能协作。

### 4. AI 分析操作

在详情页拆参数、扩词条、优化 / 翻译 / 反推提示词。

<img src="./photo/AI分析操作指引.gif" alt="AI 分析操作指引" width="860" />

**动图里在做什么：**

1. 打开任意素材详情，对照看效果图和正 / 负向提示词
2. **参数分析：** 把长提示词拆成可点的胶囊词条，方便局部替换
3. **AI 词条：** 按当前参数生成同类可替换候选
4. **优化 / 翻译 / 反推：** 整理结构、中英互译，或根据图反推完整提示词
5. 右键 AI 按钮可切换服务商、模型和规则

先在「模型配置」里填好接口；不配 AI 也能正常本地浏览和编辑。

### 5. 分类 · 标签 · 参数词库

把分类、标签和参数词养起来，素材多了也好找、好复用。

<img src="./photo/分类-标签-参数页面.gif" alt="分类标签参数页面" width="860" />

**动图里在做什么：**

1. **分类：** 分组、写说明、配图，浏览时按类筛选
2. **标签：** 统一命名，打标后全文都能搜到
3. **参数词库：** 沉淀参数名、变量和默认值，支持导入导出
4. 详情页可手改分类 / 标签，也可让 AI 建议后一键采纳

词库越齐，库越大越容易找回「那张图」和「那套词」。

### 6. 去重与压缩

扫重复图，批量压图 / 压视频，给硬盘喘口气。

<img src="./photo/去重-压缩.gif" alt="去重与压缩" width="860" />

**动图里在做什么：**

1. 进「批量管理」，看各组数量和标签概况
2. **去重：** 扫描重复图，确认后留一份、清掉多余
3. **压图：** 批量压缩，可保留原格式或转 WebP，质量可调
4. **压视频：** 批量压缩，可选目标分辨率

素材堆多了，先做这两项，体积会小不少，画质也能尽量保住。

### 7. API 模型设置

配好多个 AI 接口和模型，给分析、反推、翻译各自定规则。

<img src="./photo/API模型设置.gif" alt="API 模型设置" width="860" />

**动图里在做什么：**

1. 打开左侧「模型配置」
2. 填接口地址、模型名和 API Key，保存
3. 可为分析、扩写、优化、翻译、反推分别指定服务商和规则
4. Key 只存在本机，系统加密保存，不会打进安装包或仓库

配一次，各处都能用；需要时右键 AI 按钮还能临时换规则。

### 8. NSFW 内容分级

自动分级、默认模糊，公开场合浏览更安心。

<img src="./photo/NSFW分级.gif" alt="NSFW 内容分级" width="860" />

**动图里在做什么：**

1. 在「内容分级」里打开自动检测和默认模糊
2. 入库或批量校正时，软件会给图片打上分级
3. 瀑布流里敏感图默认模糊，详情里可临时看清或再模糊
4. 支持批量重校和速度调节

素材来源杂、或要在协作 / 公开场合看库时，会用得上。

### 9. 启动图库

换掉启动页轮播图，开场也是自己的风格。

<img src="./photo/启动图库.gif" alt="启动图库" width="860" />

**动图里在做什么：**

1. 设置里打开「启动图库」
2. 添加、替换或删除轮播图
3. 下次启动就按你的图库展示
4. 安装包默认带 6 张，可换成自己的作品

纯个性化，不动素材库和 AI 配置。

## ✨ 核心亮点

> **素言** 把效果图、正 / 负向提示词、分类标签和可替换词条放进同一套本地库，灵感能搜到、能复用、也能一键带走。

- 🗂️ **本地优先** — 素材和提示词都在本机，打开就能用，不依赖云同步
- 🖼️ **图文一体** — 效果图和提示词绑在一起看，浏览、编辑、复制、分享顺手
- ✨ **AI 可选** — 拆参数、扩词条、反推、翻译、优化，需要时再上；不配也能用
- 🧹 **好整理** — 压缩、去重、批量导入导出，库大了也不慌

- **规则预设** — 不同操作可配不同服务商 / 模型 / 规则，右键快切

## 📥 快速开始

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64 位） |
| 磁盘空间 | 按你的图片 / 视频量预留即可 |
| 网络 | 可选；用 AI、解析分享链接时才需要 |

### 安装步骤

1. **下载** 最新安装包 `素言-Setup-0.1.0.exe`，或直接用便携版里的 `素言.exe`  
   （软件本身免费；建议从 [GitHub Releases](https://github.com/guliacer/SuYan) 获取。若别处收费，请 [举报](https://github.com/guliacer/SuYan/issues)，也可发给我们，核实后会在项目中公示）
2. **安装或打开** 按向导安装，或直接跑便携版
3. **导入素材** 左侧「导入素材」：图片、粘贴、文档或分享包都行  
   （可对照 [本地导入](#1-本地素材导入) / [网络导入](#2-网络素材导入)）
4. **开始整理** 瀑布流里浏览，点进详情改提示词；要用 AI 时再配接口  
   （可对照 [AI 分析](#4-ai-分析操作) / [API 设置](#7-api-模型设置)）

## 安全说明：API 信息怎么保管

- API Key 只在软件「模型配置」里填写，保存在本机：`userData/library/ai-settings.json`
- 优先用系统 `safeStorage` 加密写成 `apiKeyEncrypted`，**不会**把明文 Key 写进仓库或安装包
- 界面侧只拿 `hasApiKey` / `apiKeyPreview`（脱敏预览），完整 Key 不会经公开设置接口返回
- 打包与 Git 已排除：`ai-settings.json`、`proxy-settings.json`、`.env*`、密钥证书和本地 library 数据
- 安装包 / 便携包是「空壳」交付：
  - 不含你的提示词组、素材图视频、收藏和词库
  - 不含 API Key、代理、窗口状态、视图 / 性能设置、本地日志
  - 只附带启动图库默认 6 张图
  - 首次启动写入空 `library.json`，其它配置按默认落在本机 `userData`
- 推 GitHub 前请确认没把本机配置、日志或 release 产物一起交上去
- 提交前可跑：`pnpm check:secrets`

## 本地开发

> 需要 Node.js 20+ 和 pnpm 11。

```bash
# 安装依赖
pnpm install

# 开发运行
pnpm dev

# 类型检查
pnpm typecheck

# 单元测试
pnpm test

# 密钥扫描
pnpm check:secrets

# 空壳打包检查（针对 release/win-unpacked）
pnpm check:empty-shell

# Windows 打包
pnpm package:win
```

产物在 `release/win-unpacked/`，安装包也在 `release/` 下。
