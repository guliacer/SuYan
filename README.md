## ✨ 核心亮点

> **素言** 把效果图、正向/负向提示词、分类标签和可替换词条收进同一套本地素材库，让灵感可检索、可复用、可一键带走。

- 🗂️ **本地优先** — 素材与提示词保存在本机，打开即用，不依赖云同步
- 🖼️ **图文一体** — 效果图与提示词绑定展示，浏览、编辑、复制、分享一气呵成
- ✨ **AI 助力** — 参数拆解、词条扩写、图像反推、翻译优化，随时辅助打磨提示词
- 🧹 **批量整理** — 压缩、去重、批量导出导入，快速清理和维护素材库

## 🚀 功能全景

### 🏠 素材浏览

- **瀑布流画廊** — 以卡片形式浏览图片与视频提示词，可调节每行列数
- **全文搜索** — 按标题、文件名、提示词内容或标签快速定位
- **筛选与排序** — 支持分类、标签、精选收藏筛选；可按导入时间、修改时间、尺寸或随机排列
- **精选收藏** — 一键标记喜爱素材，单独查看精选列表

### 📥 导入素材

- **导入图片** — 从本地批量选择图片加入素材库
- **粘贴导入** — 从剪切板直接粘贴图片，快速入库
- **导入文档** — 识别 Word 文档中的图片与对应提示词
- **导入分享** — 导入他人分享的 ZIP 素材包，或解析网络分享链接

### 🗂️ 批量管理

- **统一列表** — 集中查看全部提示词组、效果图数量与标签概况
- **批量导出 / 导入** — 按所选组导出分享包，或批量导入整理
- **批量删除** — 选中多组后一次清理，删除前二次确认
- **图像压缩** — 批量压缩本地图片，可保留原格式或转为 WebP，并调节质量
- **视频压缩** — 批量压缩视频素材，可选择目标分辨率
- **去重检测** — 扫描重复图片，保留一份后清理冗余

### 📝 提示词详情

- **图文对照编辑** — 在详情页同时查看效果图与正向 / 负向提示词
- **一键复制 / 分享** — 复制提示词到剪切板，或导出分享包带走
- **分类与标签** — 手动维护，也可让 AI 识别建议
- **模型标注** — 为素材记录生成方式，方便日后筛选与回溯
- **多图切换** — 同一组提示词可管理多张效果图，支持导入、喜爱与删除

### ✨ AI 创作助手

- **参数分析** — 把长提示词拆成可点击的莫兰迪胶囊，便于局部替换
- **AI 词条** — 针对当前参数生成同类可替换候选词
- **提示词优化** — 按场景规则整理结构，提升画面可控性
- **中英翻译** — 在中文与英文版本间忠实翻译，尽量保留参数结构
- **图像反推** — 根据效果图反推完整提示词
- **规则预设** — 为不同操作配置服务商、模型与规则，右键可快速切换

### 🎬 视频提示词

- **视频卡片** — 独立展示视频类提示词，带封面与时长信息
- **关键帧时间轴** — 一键生成关键帧，方便回顾镜头节奏
- **参考素材** — 从文件、剪切板或链接导入参考图 / 音频，辅助复现

### 📚 词库组织

- **分类浏览** — 管理分类词库，支持分组、说明与配图
- **标签浏览** — 维护可检索标签体系，便于统一命名
- **参数词库** — 沉淀 AI 分析用的参数、变量与默认值，支持导入导出

### 🌐 资源推荐

- **常用站点入口** — 汇总即梦、Civitai、LibLib 等提示词与素材站点
- **一键复制网址** — 快速复制链接，方便在浏览器中继续探索

### ⚙️ 系统设置

- **模型配置** — 添加多个 AI 接口，配置地址、模型与 API Key；可为分析、反推、翻译等操作指定规则
- **内容分级** — 自动 NSFW 分级、默认模糊、批量校正，以及分级速度调节
- **网络代理** — 系统代理 / 手动代理 / 直连切换，支持自动检测，便于解析分享与远程下载
- **启动加速** — 按硬件情况选择硬件加速模式，重启后生效
- **启动图库** — 自定义启动页轮播图
- **外观切换** — 浅色 / 深色主题一键切换

## 📥 快速开始

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64 位） |
| 磁盘空间 | 建议预留足够空间存放图片与视频素材 |
| 网络 | 可选；使用 AI 能力、分享链接解析时需要 |

### 安装步骤

1. **下载** 最新安装包 `素言-Setup-0.1.0.exe`，或直接使用便携目录中的 `素言.exe`
2. **安装 / 打开** 按向导完成安装，或直接运行便携版
3. **导入素材** 从左侧「导入素材」选择图片、粘贴、文档或分享包
4. **开始整理** 在瀑布流中浏览，打开详情编辑提示词，需要时再配置 AI 接口

## ❓ 常见问题

<details>
<summary><b>素材存在哪里？会不会上传到网上？</b></summary>

**A：** 默认保存在本机本地素材库中，日常浏览与编辑不需要联网。只有在你主动使用远程 AI、解析网络分享或下载远程图片时，才会发起网络请求。

</details>

<details>
<summary><b>如何配置 AI 能力？</b></summary>

**A：** 打开左侧「模型配置」，添加接口地址、模型和 API Key 后保存。可在详情页右键 AI 相关按钮，为不同操作选择服务商、模型与规则。

</details>

<details>
<summary><b>分享链接解析失败怎么办？</b></summary>

**A：** 先检查网络是否正常；若站点需要代理，请到「网络代理」中开启或填写代理地址，并可使用「自动检测」。

</details>

<details>
<summary><b>如何把素材分享给别人？</b></summary>

**A：** 在详情页使用「分享」，或在「批量管理」中导出所选提示词组，生成分享包后发送给对方导入。

</details>

<details>
<summary><b>NSFW 图片如何处理？</b></summary>

**A：** 在「内容分级」中可开启自动分级与默认模糊。已分级图片可在详情中临时显示或恢复模糊，也可批量重新校正。

</details>

<details>
<summary><b>视频相关功能不可用？</b></summary>

**A：** 视频卡片、关键帧与视频压缩依赖视频相关能力。请确认相关功能已启用；若仍不可用，重新安装或更新应用后再试。

</details>

## 安全说明：API 信息隔离

- API Key 只允许在软件「模型配置」中填写，保存在本机用户数据目录：`userData/library/ai-settings.json`。
- 应用会优先使用系统 `safeStorage` 加密写入 `apiKeyEncrypted`，**不会**把明文 Key 写入仓库或安装包。
- 渲染层只接收 `hasApiKey` / `apiKeyPreview`（脱敏预览），完整 Key 不会通过公开设置接口返回。
- 打包与 Git 已排除：`ai-settings.json`、`proxy-settings.json`、`.env*`、密钥证书与本地 library 数据。
- 推送 GitHub 前请确认没有把本机配置、日志或 release 产物一并提交。
- 提交前可运行：`pnpm check:secrets`。

## 本地开发

> 需要 Node.js 20+ 与 pnpm 11。

```bash
# 安装依赖
pnpm install

# 开发运行
pnpm dev

# 类型检查
pnpm typecheck

# 单元测试
pnpm test

# 密钥扫描
pnpm check:secrets

# Windows 打包
pnpm package:win
```

打包产物位于 `release/win-unpacked/` 与 `release/` 下的安装包。
